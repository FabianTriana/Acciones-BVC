# Acciones BVC:
Dash (Python microframework) web app designed as dashboard with basic information and fundamental/technical indicators of Bolsa de Valores de Colombia (Colombia's stock exchange) most traded stocks.

## Tools:
* Dash-Plotly
* Pandas
* Cufflinks
* HTML
* CSS
* Beautiful Soup
* Heroku
